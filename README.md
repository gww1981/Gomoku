# 五子棋 - 经典木纹风格

一款功能完善的五子棋网页游戏，支持双人博弈和人机对战。采用经典木纹主题，配备背景音乐、自定义背景和录像回放功能。

## 功能特点

### 核心玩法
- **双人模式**：两位玩家交替落子，适合与朋友对弈
- **人机对战**：三个难度等级，梯度分明
  - 🟢 **简单（新手级）**：会连五会堵五，但视野窄、50%概率下废棋
  - 🟡 **中等（业余级）**：攻防评分绝对最优，始终阻挡三连威胁
  - 🔴 **困难（熟练级）**：Minimax + Alpha-Beta 搜索（深度 4），走法排序优化剪枝
- **胜负判定**：横/竖/左斜/右斜任意方向连成五子即胜
- **获胜高亮**：胜利时自动流光动画亮显五子连线
- **悔棋功能**：
  - 双人模式：撤回 1 步
  - 人机模式：撤回 2 步（玩家落子 + AI 回应）
- **30 秒步时计时器**：AI 思考时暂停计时，超时自动判负
- **最后落子高亮**：最新一步动态圆环高亮，便于快速定位棋局焦点

### 录像回放
- **自动保存**：每局游戏结束自动保存录像
- **播放控制**：暂停/继续、前进/后退、跳转首尾
- **进度指示**：实时显示当前步数/总步数（如 3/15）
- **录像管理**：支持单条删除或一键清空

### 个性化设置
- **背景音乐**：内置 5 首预设音乐，支持自定义上传 MP3/WAV/OGG
- **背景图片**：4 款预设木纹/自然主题背景，支持自定义上传 JPG/PNG/WebP
- **播放控制**：播放/暂停、上一曲/下一曲、音量调节、静音

### 界面优化
- **木纹主题**：立体木框边框、多层阴影、颗粒纹理
- **棋子光泽**：径向渐变模拟真实棋子立体球面效果
- **弹性动画**：落子时弹性缩放，获胜时流光脉动
- **悬停预览**：棋盘落子时显示半透明预览棋子
- **弹窗过渡**：设置面板 fade + scale 缩放动效
- **响应式设计**：适配 320px ~ 768px+ 不同屏幕尺寸

## 技术栈

- HTML5 + CSS3 + JavaScript（纯原生，零框架依赖）
- 15×15 标准五子棋棋盘（CSS Grid 布局）
- Web Worker 异步 AI 计算（不阻塞 UI）
- localStorage 持久化存储（录像、背景、音乐设置）
- 事件发布/订阅模式（EventEmitter）解耦状态与 UI
- TDD 测试（43+ 测试覆盖 Board、AI、GameController、TimerController、EventEmitter、ReplayManager）

## AI 算法设计

### 三级难度定义

| 维度 | 🟢 简单（新手） | 🟡 中等（业余） | 🔴 困难（熟练） |
|------|---------------|---------------|---------------|
| 搜索视野 | radius=1（相邻 8 格） | radius=2（两圈内） | radius=2 |
| 连五获胜 | ✅ 100% | ✅ 100% | ✅ Minimax |
| 阻挡连五 | ✅ 100% | ✅ 100% | ✅ 100% |
| 阻挡三连 | ❌ 不管（新手特征） | ✅ 必堵 | ✅ 必堵 |
| 选子策略 | 50% 进攻分最优 + 50% 纯随机 | 攻防评分绝对最优 | 深度 4 + Alpha-Beta |
| 犯错率 | ~50% 下废棋 | 无 | 无 |
| 计算线程 | Worker 或主线程 | Worker 或主线程 | Worker（不阻塞 UI） |

### 评分规则

| 连子数 | 开放端点 | 类型 | 分值 |
|--------|----------|------|------|
| 5+ | - | 连五 | 100,000 |
| 4 | 2 | 活四 | 10,000 |
| 4 | 1 | 冲四 | 1,000 |
| 3 | 2 | 活三 | 1,000 |
| 3 | 1 | 眠三 | 100 |
| 2 | 2 | 活二 | 100 |
| 2 | 1 | 眠二 | 10 |

### 技术亮点

- **AI 评价缓存**：Memoization 缓存 evaluatePosition 结果，棋盘变更时自动失效
- **走法排序**（Move Ordering）：按启发式评分降序搜索，显著提升 Alpha-Beta 剪枝效率
- **候选生成**：boolean[][] 替代 Set<String>，消除字符串操作开销
- **Web Worker**：困难模式在独立线程计算，不阻塞 UI 渲染
- **异步回退**：浏览器不支持 Worker 时自动降级为同步计算

## 录像回放

对局结束后自动保存。在设置 → 录像 Tab 中：

- 点击录像条目开始回放
- 播放控制：⏮ 跳到开头 / ⏪ 后退一步 / ⏸ 暂停 / ▶ 继续 / ⏩ 前进一步 / ⏭ 跳到结尾
- 进度显示：当前步数/总步数
- 回放结束或点击重新开始可退出回放状态
- 支持删除单条录像或清空全部

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

或使用本地服务器：

```bash
# 使用 npx serve
npx serve .

# 或使用 Python
python -m http.server 8080
```

然后访问 http://localhost:8080

## 操作说明

1. 选择游戏模式：**双人** 或 **人机**
2. 人机模式下可通过下拉菜单选择 AI 难度（简单/中等/困难）
3. 黑方先手，点击棋盘空格落子；每次落子后下一方计时器重置为 **30s**
4. AI 思考时计时器暂停，AI 落子后恢复
5. 最新一步落子会显示动态圆环高亮，下一步时自动切换到新位置
6. 可使用 **悔棋** 回退对局（双人 1 步 / 人机 2 步）
7. 获胜时五子连线以金色流光动画高亮显示
8. 点击 **重新开始** 可随时重置游戏
9. 点击 ⚙️ 打开设置面板：
   - **背景 Tab**：选择预设背景或上传自定义图片
   - **音乐 Tab**：选择预设音乐、上传自定义音乐、控制播放
   - **录像 Tab**：查看历史对局录像、播放控制、管理录像
