# 五子棋 - 经典木纹风格

一款漂亮的五子棋网页游戏，支持双人博弈和人机对战。采用经典木纹主题，带来真实的棋盘体验。

## 功能特点

- **双人模式**：两位玩家交替落子，适合与朋友对弈
- **人机对战**：三个难度等级可选
  - 简单：随机落子，适合新手
  - 中等：启发式评分，识别威胁
  - 困难：Minimax + Alpha-Beta 剪枝算法
- **胜负判定**：横/竖/左斜/右斜方向连五即胜
- **获胜高亮**：胜利时自动亮显五子连线
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- HTML5 + CSS3 + JavaScript（纯原生，无框架依赖）
- 15×15 标准棋盘

## 项目结构

```
├── index.html        # 游戏入口页面
├── css/
│   └── style.css     # 木纹主题样式
├── js/
│   ├── game.js       # 核心游戏逻辑（Board 类、胜负判定）
│   ├── ai.js         # AI 算法（三种难度）
│   └── ui.js         # UI 交互控制
└── README.md
```

## AI 算法设计

| 难度 | 算法 | 说明 |
|------|------|------|
| 简单 | 随机落子 | 从所有空位中随机选择 |
| 中等 | 启发式评分 | 评估每个位置的攻防价值，识别活三、活二等模式 |
| 困难 | Minimax + Alpha-Beta | 深度 4 层搜索，智能剪枝优化 |

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

或使用本地服务器：

```bash
# 使用 npx
npx serve .

# 或使用 Python
python -m http.server 8080
```

然后访问 http://localhost:8080

## 操作说明

1. 选择游戏模式：双人 或 人机
2. 如选择人机模式，可进一步选择 AI 难度
3. 黑方先手，点击棋盘落子
4. 获胜时五子连线会高亮显示
5. 点击"重新开始"可重置游戏

## 游戏规则

- 黑白双方轮流落子
- 黑方先行
- 横、竖、左斜、右斜任意方向连成五子获胜
- 棋盘填满且无胜者则平局
