# 五子棋 - 经典木纹风格

一款功能完善的五子棋网页游戏，支持双人博弈和人机对战。采用经典木纹主题，配备背景音乐、自定义背景和录像回放功能。

## 功能特点

### 核心玩法
- **双人模式**：两位玩家交替落子，适合与朋友对弈
- **人机对战**：三个难度等级可选
  - 简单：随机落子，适合新手
  - 中等：启发式评分，识别活三、活二等威胁模式
  - 困难：Minimax + Alpha-Beta 剪枝算法，深度 4 层搜索
- **胜负判定**：横/竖/左斜/右斜任意方向连成五子即胜
- **获胜高亮**：胜利时自动流光动画亮显五子连线

### 个性化设置
- **背景音乐**：内置预设音乐，支持自定义上传 MP3/WAV/OGG
- **背景图片**：4 款预设木纹主题背景，支持自定义上传 JPG/PNG/WebP
- **播放控制**：播放/暂停、上一曲/下一曲、音量调节、静音

### 录像回放
- **自动保存**：每局游戏结束自动保存录像
- **历史列表**：按时间倒序展示，最多保存 20 条
- **逐步回放**：800ms 间隔自动播放完整对局过程
- **录像管理**：支持单条删除或一键清空

### 界面优化
- **木纹主题**：立体木框边框、多层阴影、颗粒纹理
- **棋子光泽**：径向渐变模拟真实棋子立体球面效果
- **弹性动画**：落子时弹性缩放，获胜时流光脉动
- **悬停预览**：棋盘落子时显示半透明预览棋子
- **弹窗过渡**：设置面板 fade + scale 缩放动效
- **响应式设计**：适配 320px ~ 768px+ 不同屏幕尺寸

## 技术栈

- HTML5 + CSS3 + JavaScript（纯原生，零框架依赖）
- 15×15 标准五子棋棋盘
- CSS Grid 棋盘布局
- localStorage 持久化存储（录像、背景、音乐设置）
- CSS 自定义属性（CSS Variables）主题系统

## AI 算法设计

| 难度 | 算法 | 搜索策略 | 说明 |
|------|------|----------|------|
| 简单 | 随机落子 | 无 | 从所有空位中随机选择 |
| 中等 | 启发式评分 | 全局空位 | 评估每个位置的攻防价值，对手威胁加权 1.1 倍优先防守 |
| 困难 | Minimax + Alpha-Beta | 候选位置 + 深度 4 层 | 仅搜索已有棋子周围 2 格范围，剪枝优化搜索效率 |

### 评分规则

| 连子数 | 开放端点 | 类型 | 分值 |
|--------|----------|------|------|
| 5+ | - | 连五 | 100,000 |
| 4 | 2 | 活四 | 10,000 |
| 4 | 1 | 冲四 | 1,000 |
| 3 | 2 | 活三 | 1,000 |
| 3 | 1 | 眠三 | 100 |
| 2 | 2 | 活二 | 100 |
| 2 | 1 | 眠二 | 10 |

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

或使用本地服务器：

```bash
# 使用 npx serve
npx serve .

# 或使用 Python
python -m http.server 8080
```

然后访问 http://localhost:8080

## 操作说明

1. 选择游戏模式：**双人** 或 **人机**
2. 人机模式下可通过下拉菜单选择 AI 难度（简单/中等/困难）
3. 黑方先手，点击棋盘空格落子
4. 获胜时五子连线以金色流光动画高亮显示
5. 点击 **重新开始** 可随时重置游戏
6. 点击 ⚙️ 打开设置面板：
   - **背景 Tab**：选择预设背景或上传自定义图片
   - **音乐 Tab**：选择预设音乐、上传自定义音乐、控制播放
   - **录像 Tab**：查看历史对局录像、回放或删除

## 录像回放

对局结束后自动保存。在设置 → 录像 Tab 中：

- 点击录像条目开始回放
- 回放时棋盘逐步重现对局过程
- 回放结束或点击重新开始可退出回放状态
- 支持删除单条录像或清空全部

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。

需要启用的浏览器功能：
- CSS Grid
- CSS Custom Properties
- localStorage
- Web Audio API（音乐功能）
