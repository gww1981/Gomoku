# 五子棋背景音乐功能设计

**日期**：2026-05-08
**功能**：背景音乐切换
**状态**：已批准

---

## 概述

为五子棋游戏添加背景音乐功能，用户可以播放 5 首预设音乐，也可以本地上传自定义音乐。

---

## 用户需求

- 预设 5 首背景音乐（来自免费音乐库）
- 支持本地上传背景音乐
- 完整播放控制（播放/暂停，切歌、音量调节、静音）

---

## 设计决策

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 预设来源 | 免费音乐库 URL | 无需用户提供文件 |
| 入口方式 | 复用设置弹窗 + Tab 切换 | 复用现有 UI，保持简洁 |
| 播放控制 | 完整控制 | 用户需要播放/暂停/切歌/音量/静音 |
| 存储 | localStorage | 保存音量、当前音乐、静音状态 |

---

## UI 结构

```
设置弹窗
├── 背景 Tab（原背景切换功能）
└── 音乐 Tab（新增）
    ├── 预设音乐列表（5首）
    ├── 当前播放曲目显示
    ├── 播放控制：◀ ▶ 🔊 ───
    └── 本地上传按钮
```

## 技术实现

### 文件修改

| 文件 | 修改内容 |
|------|----------|
| `index.html` | 添加音乐 Tab 切换结构 |
| `css/style.css` | 添加音乐面板样式、播放控制样式 |
| `js/game.js` | 添加音频管理函数（播放/暂停/切歌/音量） |
| `js/ui.js` | 添加音乐 Tab 交互逻辑 |

### 数据存储

- **存储位置**：`localStorage`
- **存储 Key**：
  - `gomoku_music_volume`：音量值（0-1）
  - `gomoku_music_muted`：静音状态（true/false）
  - `gomoku_music_current`：当前播放音乐（preset-1~5 或 custom）

### 预设音乐

使用免费音乐库 URL（待搜索确认）：
1. 预设音乐 A
2. 预设音乐 B

4. 预设音乐 D
5. 预设音乐 E

### 本地音乐限制

- 单文件最大 10MB
- 格式支持：MP3、WAV、OGG
- 新上传会覆盖之前的自定义音乐

## 交互流程

1. 点击设置按钮 → 弹窗显示
2. 点击"音乐" Tab 切换到音乐面板
3. 点击预设音乐 → 开始播放
4. 使用播放控制：
   - ▶/⏸ 播放/暂停
   - ◀/▶ 上一首/下一首
   - 🔊 音量滑块
   - 🔇 静音开关
5. 点击"选择文件"上传本地音乐
6. 关闭弹窗 → 记住当前播放状态

- 静音开关
- 当前播放曲目名称

- 标准文件输入样式
- 支持点击上传
