# 五子棋背景切换功能设计

**日期**：2026-05-08
**功能**：背景切换器
**状态**：已批准

---

## 概述

为五子棋游戏添加背景切换功能，用户可以从 4 张预设木质纹理背景中选择，也可以本地上传自定义背景图片。

---

## 用户需求

- 预设 4 张木质纹理背景图片
- 支持本地上传背景图片
- 保持与棋盘木纹风格的协调

---

## 设计决策

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 预设风格 | 木质纹理 | 与棋盘木纹风格统一 |
| 本地图片适配 | cover（拉伸填充） | 完整覆盖且不变形 |
| 交互方式 | 设置按钮 + 弹窗 | 直观易用 |
| 实现方案 | 混合方案 | 平衡离线能力与文件大小 |

---

## UI 结构

```
[设置按钮] → [模态弹窗]
              ├── 预设背景（2×2 网格）
              │   ├── 木纹 A（深棕）
              │   ├── 木纹 B（浅棕）
              │   ├── 木纹 C（红木）
              │   └── 木纹 D（竹色）
              ├── 本地上传区域
              │   └── [选择文件] 按钮
              └── 当前已选提示
```

---

## 技术实现

### 文件修改

| 文件 | 修改内容 |
|------|----------|
| `index.html` | 添加设置按钮和模态框结构 |
| `css/style.css` | 添加模态框、背景选择器样式 |
| `js/ui.js` | 添加设置弹窗交互逻辑 |
| `js/game.js` | 添加背景应用函数 |

### 数据存储

- **存储位置**：`localStorage`
- **存储 Key**：`gomoku_background`
- **存储值**：
  - 预设背景：`"preset-1"`, `"preset-2"`, `"preset-3"`, `"preset-4"`
  - 自定义背景：Base64 DataURL

### 预设背景

使用网络木质纹理图片（CDN URL）：
1. 深棕色木质
2. 浅棕色木质
3. 红木质感
4. 竹色/浅木色

### 本地图片限制

- 单张图片最大 2MB
- 限制保存 1 张自定义背景（新上传会覆盖）
- 格式支持：JPG、PNG、WebP

---

## 交互流程

1. 用户点击设置按钮 → 弹出模态框
2. 点击预设背景 → 立即应用，显示选中态
3. 点击"选择文件" → 文件选择器 → 选中的图片作为背景
4. 关闭弹窗 → 记住用户选择
5. 页面加载时 → 自动恢复上次选择的背景

---

## 组件清单

### 设置按钮
- 位置：顶部控制栏，模式切换旁边
- 图标：齿轮/扳手图标
- 状态：默认、悬停

### 模态弹窗
- 居中显示
- 半透明黑色遮罩背景
- 点击遮罩可关闭
- 包含关闭按钮

### 背景网格
- 2×2 布局
- 每个格子显示背景缩略图
- 选中状态：金色边框 + 勾选标记

### 上传按钮
- 标准文件输入样式
- 支持点击和拖拽上传

---

## 验收标准

1. ✅ 4 张预设木质纹理背景可选择
2. ✅ 本地图片可上传并显示
3. ✅ 刷新页面后背景选择被记住
4. ✅ 弹窗可正常打开和关闭
5. ✅ 背景覆盖棋盘区域，不影响游戏功能
